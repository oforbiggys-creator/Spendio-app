# Spendio

Spendio is a fintech platform focused on improving financial access in Africa, starting with Nigeria.

## Vision
To provide smart payments, transparent micro-loans, and financial tools that empower individuals and small businesses.

## Core Features (MVP)
- User registration & login
- Wallet system
- Loan application & tracking
- Interest-based repayment
- Transaction history
- Admin loan approval (future)
