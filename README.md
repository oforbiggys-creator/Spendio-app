# Spendio

Spendio is a fintech platform focused on improving financial access in Africa, starting with Nigeria.

## Vision
To provide simple, trusted financial tools — payments, wallets, and loans — that empower individuals and small businesses.

## Current Features
- Home screen
- User registration (mock)
- Login flow (mock)

## Next Phase
- Dashboard & wallet
- Loan request and repayment system
- Payment gateway integration
- KYC & security layers
